Hi, I'm Leigh.

I'm a full-stack web developer and recent graduate of General Assembly's Web Development Immersive bootcamp, a twelve-week program that emphasizes object oriented programming, MVC framework, data modeling, test-driven development, and agile workflow.

I decided to become a web developer because I love the balance that it strikes between logic and creativity and I love being able to create, solve problems, and learn new things every single day.

I particularly enjoy working on the front-end as it gives me the opportunity to express my creativity. I am currently looking for a junior front-end or full-stack job opportunity in Portland, Oregon or San Francisco, California.

Prior to taking the plunge into web development, I earned a B.A. in economics and worked in communications for an environmental nonprofit for three years.

Sites & Apps

Triply

Triply, a travel recommendation app, is the first full-stack web application that I built from scratch. I did all of the front-end and back-end work on this application. The app is built in Ruby on Rails and utilizes Bootstrap and Sass on the front end. It features a hand rolled user authentication system that allows users to create, read, update, and delete travel recommendations.
