About Me

Hello, I'm Leigh - a full-stack web developer and recent graduate of General Assembly's Web Development Immersive bootcamp, a twelve-week program that emphasizes object oriented programming, MVC framework, data modeling, test-driven development, and agile workflow. I decided to become a web developer because I love the balance that it strikes between logic and creativity and I love being able to create, solve problems, and learn new things every single day. Prior to taking the plunge into web development, I studied economics undergrad and worked in communications for an environmental nonprofit for three years.
